Atividades de Banco de Dados 3º Trimestre

Link do Documento

Nessa atividade praticamos os comando DDL e DML do SQL. Criamos um banco de dados para uma biblioteca e realizamos difersas buscas e alterações nas tabelas.

Competências e Habilidades: H4

O objetivo dessa atividade era conhecer os últimos comandos SQL da disciplina e conhecer um pouco da interface do PgAdmin, contruindo uma bando de dados para gerenciar pedidos de uma loja.
